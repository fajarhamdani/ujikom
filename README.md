# Aplikasi To-Do List

## Deskripsi

Aplikasi sederhana berbasis PHP untuk mencatat tgas harian

## Fitur

-   Tambah Tugas
-   Tandai Tugas Selesai
-   Hapus Tugas

## Struktur Folder

-   'app/Models/Task.php' - Model yang digunakan
-   'app/Http/Controllers/ToDoListController.php' - Controller Utama
-   'resources/views/to_do_list/indext.blade.php' - Halaman Utama

## Cara Menjalankan

1. Salin projek ke folder /var/www/html (untuk Ubuntu dan menggunakan Apache)
2. Install database MySql
3. Buat database ujikom
4. Jalankan file migrasi
